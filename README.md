# MacrowEx

MacrowEx provide DSL for define rules of text replacing.

It's inspired by https://github.com/syguer/macrow Ruby gem.

## Installation

If [available in Hex](https://hex.pm/docs/publish), the package can be installed

```elixir
def deps do
  [
    {:macrow_ex, "~> 0.1.0"}
  ]
end
```
